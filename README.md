# Marstek Venus – Home Assistant Skript

Python- / AppDaemon-Skript zur Steuerung von Marstek-Venus-Batteriesystemen
in Home Assistant über Modbus.

## Funktionen
- Dynamische Lade- und Entladesteuerung
- SoC-Grenzen
- Notfall-Lademodus
- Debug- und Detail-Logging

## Voraussetzungen
- Home Assistant
- AppDaemon
- Modbus (TCP oder RTU)
